## Project Description
This Android application allows users to book medical appointments easily.  
The application contains four screens:

1. Splash Screen
2. Main Screen
3. Book Appointment Screen
4. Confirmation Screen

## Features

• Splash screen with application logo  
• Home screen with welcome message  
• Appointment booking form  
• Date and time picker dialogs  
• Gender selection using RadioGroup  
• Appointment type selection using Spinner  
• Form validation  
• Confirmation screen displaying booking details  

## Technologies Used

• Kotlin  
• Android Studio  
• XML Layout Design  

## Application Flow

Splash Screen → Main Screen → Book Appointment Screen → Confirmation Screen

## UI Enhancements

• Clean and simple user interface  
• Smooth screen navigation  
• Basic animations

## How to Run

1. Clone the repository
2. Open project in Android Studio
3. Build and run on emulator or Android device
